## THIS IS EXAMPLE OF USING PYCAI2 FOR A BOT DISCORD

# Instalation
```
pip install PyCAI2 discord
```
then git clone this repo
```
git clone
```

# settings 
```
token_discord = "<DISCORD BOT TOKEN>"

owner_id = '<CAI TOKEN>'
char = "<CHAR ID>"
chat_id = "<CHAT ID>"

aut_set ={
    "author_id": "<AUTHOR ID>",
    "is_human": True,
    "name": "<CAI USER NAME>"
}
```

# RUN
```
python main.py
```
